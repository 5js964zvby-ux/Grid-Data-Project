GRID DATA PROJECT - VERSION 5 FOCUS ZOOM EXPERIMENT

WHAT CHANGED
- Clicking any workflow stage now opens a zoom-style focus view.
- The focus view expands from the clicked card instead of snapping down the page.
- Input, process, output, and engineering value appear inside the focused stage view.
- Previous/Next buttons, clickable progress dots, and keyboard arrows move through the six stages.
- Escape, the close button, or clicking the dark backdrop returns to the workflow.
- The old snap-down summary card is removed from the normal page flow.
- On phones, focus mode becomes a full-screen guided view.

UPLOAD
1. Open the Grid-Data-Project repository.
2. Choose Add file > Upload files.
3. Upload index.html, style.css, script.js, and the entire assets folder.
4. Replace the existing files and commit.
5. Wait 1-3 minutes.
6. Refresh with Ctrl+Shift+R or Ctrl+F5.

ROLLBACK
Your Version 4.1 ZIP remains your stable snap-down version. Re-upload it at any time to restore the old interaction.
