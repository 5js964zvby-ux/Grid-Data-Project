UPLOAD
1. Open the Grid-Data-Project repository.
2. Choose Add file > Upload files.
3. Upload index.html, style.css, script.js, and the entire assets folder.
4. Replace the existing files and commit the changes.
5. Wait 1-3 minutes.
6. Refresh with Ctrl+Shift+R on the laptop.
7. On iPhone, close the previous Safari tab and reopen the clean URL.
