- The graph now has a built-in fallback: if the lightbox script does not load,
  clicking it opens the full-resolution image in a new tab.
- Changes the redundant header Explore button to View poster.
- Keeps the hero Explore the workflow button unchanged.

UPLOAD
1. Open the Grid-Data-Project repository.
2. Choose Add file > Upload files.
3. Upload index.html, style.css, script.js, and the entire assets folder.
4. Allow GitHub to replace the existing files.
5. Commit the changes.
6. Wait 1-3 minutes.
7. Refresh the laptop page with Ctrl+Shift+R or Ctrl+F5.
