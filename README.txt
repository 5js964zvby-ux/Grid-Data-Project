GRID DATA PROJECT - VERSION 5.1: STORY START + FOCUS POLISH

LANDING BEHAVIOR
- The public website now always opens at the hero/title.
- Internal navigation scrolls smoothly without leaving #workflow or another section
  in the URL, so refreshing the page returns to the beginning.
- The final QR code should use the clean URL with no # section suffix.

FOCUS MODE UPGRADES
- Workflow cards assemble once when they enter the screen.
- The selected card briefly enlarges before the zoom view opens.
- The connecting workflow line advances to the active stage.
- Each stage has a unique process heading.
- Input, Process, Output, and Engineering Value have small scanning icons.
- The stage panel and information cards animate together.
- Visitors can swipe left or right between stages on mobile.
- The focus panel resets to the top whenever it opens.

UPLOAD
1. Open the Grid-Data-Project GitHub repository.
2. Choose Add file > Upload files.
3. Upload index.html, style.css, script.js, and the entire assets folder.
4. Replace the existing files and commit.
5. Wait 1-3 minutes.
6. Refresh with Ctrl+Shift+R on the laptop.

Use this clean URL for testing and eventually for the QR code:
