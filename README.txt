FILES TO UPLOAD
- index.html
- style.css
- script.js
- assets/ folder

HOW TO UPDATE YOUR CURRENT GITHUB SITE
1. Open your Grid-Data-Project repository.
2. Click Add file > Upload files.
3. Upload the three main files and the assets folder.
4. Allow index.html, style.css, and script.js to replace the existing versions.
5. Click Commit changes.
6. Wait about 1–3 minutes and refresh the website.

REQUIRED PERSONALIZATION
1. Open index.html in Notepad.
2. Replace BOTH occurrences of:
   YOUR_LINKEDIN_URL
   with your actual LinkedIn profile URL.

PUBLIC-SAFETY CHECK
Everything uploaded to GitHub Pages is public. Do not upload:

- Proprietary code
- Internal filenames or folders
- Anything not approved by your supervisor
